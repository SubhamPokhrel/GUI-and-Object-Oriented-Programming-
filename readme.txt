README 

Purpose:
  This program allows user to manage a schedule for classes
	o add/remove course
	o add/change instructor
	o get/set course prefix
	o get/set course section
	o get/set course number
	o get/set course name
	o set instructor
	o View all courses
	o View all courses of a prefix
	o view all sections of a course
	o add/remove students to a course
	o view students in a course (in progress)

Programming topics to note:
	simple GUI programming
	Object oriented programming
	operator overloading
